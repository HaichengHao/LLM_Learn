## 构建图的流程    
0. 需要的包 
```python
from langgraph.graph import StateGraph,START,END
from typing import TypedDict

```
1. 创建状态schema state_schema
```python

class GraphState(TypedDict):
    process_data:dict


```
2. 创建节点函数,传入状态继承与state_schema
```python
def input_node(state:GraphState)->GraphState:
    print(f'input_node节点执行state.get("process_data")方法的结果:{state.get("process_data")}')
    return {
        'process_data':{
            'input':"input_value"
        }
    }

def process_node(state:dict)->dict:
    print(f'process_node节点执行state.get("process_data")方法的结果:{state.get("process_data")}')
    return {
        'process_data':{
            "process":"process_value9527"
        }
    }

def output_node(state:GraphState)->GraphState:
    print(f'output_node节点执行state.get("process_data")方法的结果:{state.get("process_data")}')
    return {
        "process_data":state.get("process_data")
    }
```
3. 创建graph(或者叫builder),传入state_schema
```python
 graph=StateGraph(GraphState)

```
4. 给graph(或者叫builder)添加节点  
```python
graph.add_node("input_node",input_node)
graph.add_node("process_node",process_node)
graph.add_node("output_node",output_node)

```
5. 给graph(或者叫builder)添加边
```python
graph.add_edge(START,"input_node")
graph.add_edge("input_node","process_node")
graph.add_edge("process_node","output_node")
graph.add_edge("output_node",END)

```
6. 编译成图
```python
app = graph.compile()
```
7. 调用
```python
result = app.invoke(
    {
        "process_data":{
            'name':'测试数据',
            'value':123321
        }
    }
)

print(f"结果{result}")
```

8. 输出图的可视化结构
```python
print(app.get_graph().print_ascii())
print(app.get_graph().draw_mermaid())
output_path = "../demoimgs/"+str(uuid.uuid4())[:8]+'.png'
with open(output_path,'wb') as f:
    f.write(app.get_graph().draw_mermaid_png(max_retries=2,retry_delay=2.0))

print(f'保存图成功,路径{output_path}')
```
