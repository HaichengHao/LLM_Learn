# deepseek用法

## 接口文档

[deepseek接口文档](https://api-docs.deepseek.com/zh-cn/)

## 用法示例

和OpenAI一样的,只需要换url和key

```python
# Please install OpenAI SDK first: `pip3 install openai`
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ.get('DEEPSEEK_API_KEY'),
    base_url="https://api.deepseek.com")

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "system", "content": "You are a helpful assistant"},
        {"role": "user", "content": "Hello"},
    ],
    stream=False
)

print(response.choices[0].message.content)
```

## LangChain表达式(LCEL)
LCEL(LangChainExpressionLanguage),其实他的用处就是使用"|"(linux中我们称其为管道符)运算符
链接LangChain应用的各个组件    

它是一种声明式的方法来链接LangChain的组件。LCEL从第一天起就被设计为支持将原型投入生产，
无需代码更改，从最简单的“提示词+大语言模型”链到最复杂的链(我们看到有人成功在生产环境中运行包含数百个步骤的LCEL链)
